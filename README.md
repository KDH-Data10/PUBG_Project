# PUBG Match Data Analysis

## Project Description

This is a personal data analysis project using PUBG API to collect match and player statistics. 
The goal is to analyze gameplay patterns, win rates, and performance metrics for learning and research purposes.

## Features

- Collect recent match IDs and player IDs from PUBG API
- Fetch detailed match statistics and telemetry data
- Analyze win rates, kills, damage, and other performance metrics
- Prepare datasets for statistical analysis and machine learning experiments

## Target Audience

This project is for personal learning and research purposes only.
There are no external users or public deployment.

## Data Source

- PUBG API is used to fetch match IDs, player IDs, and match details
- Data is collected for personal analysis only
- No commercial use or monetization

## Development Status

- Project is in the early development phase
- Scripts for data collection and initial analysis are available in this repository
- Planning to expand dataset and perform statistical analysis and machine learning

## Usage

1. Set your PUBG API key in `config.py`
2. Run `collect_data.py` to fetch recent match data
3. Use `analysis.ipynb` to explore and analyze collected data
